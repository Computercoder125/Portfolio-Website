They first have to create an account (shown in the left picture), and after that they are taken to the main webpage where they can book an appointment, search for tutors, or delete an already booked appointment. Hangman



The goal of this game is to provide a fun yet enriching experience of players to capitalize on their knowledge of states. This project was made in the C language during my senior year of high school. It emulates a classic hangman game yet instead of drawing the figure, it decrements the amount of users' lives each time he/she guesses an incorrect letter. Passcode



The purpose of this Guided User Interface Program is to emulate an iphone's passcode screen. It asks the user to Enter a 4 digit passcode, and if the passcode is correct, it displays an empty screen (analagous to the home screen of an iphone). If not, the user is prompted, "Wrong passcode. Try Again". If you want to try it out, feel free to click the link below!
